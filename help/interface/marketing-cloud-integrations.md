# Integrações da Experience Cloud

A Adobe Experience Cloud é um conjunto abrangente das melhores soluções integradas disponíveis no setor, que aproveita a plataforma de dados comum com um conjunto dos principais serviços avançados.

## Principais serviços - Ativar suas soluções para os principais serviços {#section_A3D024994DA3492F8435CFCC4EF035C2}

Ajuda: [Ativar suas soluções para os principais serviços](core-services/core-services.md#concept_07ED1D5C64234E77976E6D572E78FB9C)

Descreve como:

* Provisionar a empresa na Experience Cloud.
* Permitir que você se torne um administrador.
* [Implementar o serviço da Experience Cloud ID](https://marketing.adobe.com/resources/help/en_US/mcvid/).
* Modernizar as implementações do Analytics e do Target por DTM.
* Começar a usar os principais serviços.

Soluções ou serviços:

* Ativação - Gerenciamento dinâmico de tags ou Launch da Adobe
* Analytics
* Target
* [Serviço da Experience Cloud ID](https://marketing.adobe.com/resources/help/en_US/mcvid/)

## Serviço da Experience Cloud ID {#section_6ECCCFA2D84D4D4F88C879C799CA9D78}

Ajuda: [Serviço da Experience Cloud ID](https://marketing.adobe.com/resources/help/en_US/mcvid/)

O serviço de ID fornece uma ID contínua e universal que identifica os visitantes por todas as soluções na Experience Cloud. Ele pode substituir o código de geração de ID para serviços como o Analytics, Audience Manager, Target, pulsação de vídeo e outros recursos e soluções da Experience Cloud.

**Soluções ou serviços aplicáveis**

* [Analytics](https://marketing.adobe.com/resources/help/en_US/mcvid/mcvid-setup-analytics.html)
* [Target](https://marketing.adobe.com/resources/help/en_US/mcvid/mcvid-setup-target.html)
* [Data Workbench](https://marketing.adobe.com/resources/help/en_US/mcvid/mcvid-dwb.html)

## Públicos-alvo{#section_5F60D7B0833348B9A1D74663AADCB42C}

Ajuda: [Públicos-alvo](audience-library/audience-library.md#topic_679810123CAA4E0CA4FA3417FB0100C7)

Criar e gerenciar públicos-alvo na biblioteca de público-alvo da Experience Cloud. Os públicos-alvo podem ser criados ou derivados de várias fontes, como:

* Novos criados no [!DNL Experience Cloud].
* Em [!DNL Analytics] segmentos publicados no [!DNL Experience Cloud].
* De [!DNL Audience Manager].

**Soluções ou serviços aplicáveis**

* [Atividades](https://marketing.adobe.com/resources/help/en_US/target/target/c_activities.html) no Adobe Target
* [Segmentação](https://marketing.adobe.com/resources/help/en_US/analytics/audiences/aam-analytics-segments.html) no Audience Manager
* [Media Optimizer](https://enterprise.efrontier.com/CMDashboard/help/internal/concepts_and_features/media_optimizer_integration_with_adobe_marketing_cloud.htm) (requer logon)

## Atributos do cliente {#section_6A9EA6847F654F129381869E5016626C}

Ajuda: [Atributos do cliente](attributes/attributes.md#concept_ACFEE7C8B8E94875BA0825CDF4913AF1)

Se você capturar os dados de clientes de empresas em um banco de dados de gerenciamento de relacionamento com o cliente (CRM), poderá fazer upload dos dados em uma fonte de dados do atributo do cliente na Experience Cloud. ## Advertising Cloud {#section_9B1935F8BBC147C89C6DB68A35CB1BAB}

Ajuda (requer logon): [Integração com as soluções e os principais serviços da Adobe Experience Cloud](https://enterprise.efrontier.com/CMDashboard/help/internal/concepts_and_features/media_optimizer_integration_with_adobe_marketing_cloud.htm)

**Soluções ou serviços aplicáveis**

**Analytics:** pode enviar dados de participação e conversão do site para o Media Optimizer, onde os dados estão disponíveis para otimização de anúncios e relatórios. Além disso, o Media Optimizer pode enviar dados de tráfego do mecanismo de busca e de redes sociais para o Analytics, no qual os dados estão disponíveis nos recursos de relatórios e análises, construtor de relatórios e Ad Hoc Analysis.

**Gerenciador dinâmico de tags:** você pode [usar o Gerenciador dinâmico de tags para criar tags de rastreamento de conversão com base em pixels do Media Optimizer](https://marketing.adobe.com/resources/help/en_US/dtm/t_media_opt_rule.html), bem como tags de rastreamento de terceiros, para as páginas de aterrissagem de pesquisa, páginas sociais e exibição de anúncios. (Você também pode criar tags do Media Optimizer diretamente no Media Optimizer.)

**Públicos-alvo da Experience Cloud:** (anunciantes com gerenciamento de exibição) você pode usar qualquer um dos [públicos-alvo da Adobe Experience Cloud](https://marketing.adobe.com/resources/help/en_US/mcloud/audience_library.html) como alvos dos anúncios de exibição. É possível usar automaticamente os públicos-alvo criados na Adobe Experience Cloud e os do Analytics publicados na Adobe Experience Cloud, bem como usar os públicos-alvo do Audience Manager quando a conta do Media Optimizer estiver configurada para permitir isso. Entre em contato com o gerente de conta para obter mais informações sobre o acesso à Adobe Experience Cloud e os principais serviços de perfis e públicos-alvo, bem como sobre a configuração inicial entre os públicos-alvo do Media Optimizer e da Adobe Experience Cloud. **Observação:** se você também usar o Adobe Target, qualquer um desses públicos-alvo publicados na Adobe Experience Cloud estarão disponíveis para atividades no Target.

**Ativos da Experience Cloud:** (anunciantes com gerenciamento de exibição) você pode usar qualquer um dos ativos da Adobe Experience Cloud como os elementos de criação dos anúncios de exibição usando a Nova visualização beta. Você deve estar [conectado ao Media Optimizer pela Adobe Experience Cloud](https://enterprise-test.efrontier.com/CMDashboard/help/internal/getting_started/t_log_in_from_adobe_marketing_cloud.htm) para acessar os ativos da Adobe Experience Cloud. Entre em contato com o gerente da conta para obter mais informações sobre como acessar a Adobe Experience Cloud.

**Notificações da Experience Cloud:** a partir do link de notificações na parte superior de cada página, você pode exibir todos os alertas gerados a partir dos modelos beta de alertas de pesquisa, bem como atualizações, publicações, menções e ativos compartilhados da Adobe Experience Cloud. Você deve estar [conectado ao Media Optimizer pela Adobe Experience Cloud](https://enterprise-test.efrontier.com/CMDashboard/help/internal/getting_started/t_log_in_from_adobe_marketing_cloud.htm) para acessar as notificações. Entre em contato com o gerente da conta para obter mais informações sobre como acessar a Adobe Experience Cloud.
